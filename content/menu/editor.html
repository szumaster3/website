Cache Editor


Overview

A powerful new tool for managing the cache, written in Kotlin.

Available tools:

  • Item editor
  • NPC editor
  • Object editor
  • Sprite editor
  • File manager

Status

This is an alpha release. Some features are still incomplete or experimental.
Currently, only manual downloads are available.
Suggestions, feedback, and bug reports are welcome.

Note: To create an issue or feature request, you must have a GitLab account.
Report a bug: Submit an issue
Request a feature: Submit a feature request


Download

  • Latest release: Editor-2.0.7-alpha+internal.zip
  • Legacy release: https://github.com/szumaster3/530-Editor/releases

Note

This project is not intended for commercial use. It is developed primarily for learning purposes and to explore and demonstrate techniques that can be applied to RSPS development challenges.
