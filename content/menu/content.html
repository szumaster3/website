Singleplayer (Offline Mode)


Singleplayer package is a fully self-contained, cross-platform release of the RuneScape experience. It allows you to play, test, and develop completely offline - without connecting to any online servers. Perfect for modding, experimenting, or simply reliving the old-school RuneScape experience privately.


Setup

  1. Download the Singleplayer package and extract it anywhere on your computer.
  2. Avoid installing into paths that contain spaces (e.g. C:\Users\My Name\Documents\) - this can cause startup issues. Prefer simple paths like C:\Singleplayer or your Desktop.
  3. Inside the extracted folder, you'll find both a server and a client directory.
  4. To start the server, run run-server.bat (Windows) or ./run-server.sh (Linux/Mac).
  5. Once the console displays 'Server is listening on port 43594', launch the client using run-client.bat or ./run-client.sh.
  6. Log in with any username and password - a new account will be created automatically.

Configuration

  • The main configuration file is located at game/worldprops/default.conf.
  • No internet connection is required once setup is complete.
  • By default, the experience rate (exp stage) is set to 1.0x. To change this after creating your character, open your save file located in data/players/<your_username>.json and edit the exp_stage value manually.

Resources

The following files are included in the resources directory:

  • plugins.zip – contains all available plugins.
  • plugins/ – extracted plugins directory.
  • commands.txt – list of commands available to plugins.
  • legacy-client.zip – the legacy client (plugins do not work here).

Plugins

  1. Unpack the desired plugin from plugins.zip (or use a folder from plugins/).
  2. Move the plugin folder (e.g., RememberMyLogin) into game/plugins/.
  3. Launch the client — the plugin will be loaded automatically.

Notes

  • This release is designed to work out of the box - any internal configuration changes are at your own risk.
  • If you modify server configuration files and things start acting strange, please revert your changes.
  • Program Files or protected system directories are not recommended installation paths.

With this setup, you can play, develop, and explore RuneScape entirely offline - preserving the 2009 RuneScape experience forever.
